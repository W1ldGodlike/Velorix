# Горячие клавиши (глобально в главном окне)

Сочетания по умолчанию ниже; их можно **переназначить** в **Настройки → раздел «Горячие клавиши»** (ТЗ §4.6). После сохранения настроек новые комбинации применяются сразу.

Подключены программно (без проблемной разметки `Invoked` в XAML):

| Сочетание | Действие |
|-----------|----------|
| Ctrl+, (клавиша запятой) | Настройки |
| Ctrl+Shift+T | Окно терминала |
| Ctrl+H | История задач |
| Ctrl+Shift+K | База знаний (Markdown, статьи из папки `Help/`) |
| Ctrl+1 | Вкладка «Скачивание (yt‑dlp)» |
| Ctrl+2 | Вкладка «Обработка (ffmpeg)» |
| Ctrl+M | Mini Player (компактная панель поверх задач из очереди) |

Меню **Инструменты → Инспектор медиа (ffprobe)** открывает отдельное окно ffprobe без переключения вкладки.
