# Извлечение кадров

Меню **Инструменты → Извлечь кадры** открывает окно упрощённого сценария §7.6:

- **Каждые N секунд** — фильтр `fps=1/N`.
- **Всего K кадров равномерно** — нужна длительность из ffprobe; задаётся `fps = K/длительность`.
- **Один кадр** — `-ss` и один выходной файл с меткой времени в имени.

Форматы: JPG, PNG, WebP. Файлы сохраняются в выбранную папку (`flux_frame_*.jpg` или один файл для режима «один кадр»).

Полный ТЗ‑набор (прогресс‑бар, ручные метки через запятую, спрайты §7.5) в этом окне не реализован.
