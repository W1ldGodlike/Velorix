# Снимок кадра из превью

В редакторе на панели над превью есть кнопка **снимка кадра**. Она сохраняет картинку из **текущего положения** воспроизведения.

## Формат

Перед сохранением можно выбрать **PNG** или **JPEG** в той же панели, где кнопка снимка.

## Куда сохраняется

Откроется обычный диалог «Сохранить как» — выберите папку и имя файла.

## Отличие от пакетного извлечения

Пакетное «вытащить много кадров по расписанию» из меню **пока не подключено** — это отдельный сценарий из плана продукта. Сейчас доступен быстрый одиночный снимок из превью.
