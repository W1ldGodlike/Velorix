# Export rail: presets and format

On the right side of the editor you will find a **collapsible rail** with steps: video, audio, filters, presets, output format, and a command preview.

## Presets

At the top of the list you will see **ready-made profiles for popular destinations** (short clips, long uploads, messengers, etc.) — each has a short hint. Below that you can store **your own** presets; they persist between launches.

## Format

Pick the file container (for example MP4 for wide compatibility, or MKV when you need extra track headroom), picture, and sound settings. Change quality numbers in small steps and watch the preview command at the bottom.

## Two-pass mode

For some codec and bitrate combinations you can enable **two-pass** encoding — slower, but steadier file size.

## Frame snapshot

The snapshot button saves an image from the current preview frame. More detail — [extract-frames.md](extract-frames.md).
