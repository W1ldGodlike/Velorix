# Main window tabs

![Three-tab diagram](assets/workspace-tabs-diagram.svg)

## Editor

Open a **local file**, watch preview, mark start and end of a clip, and run export to a new file. A collapsible settings rail sits on the right.

## Downloads

Work with **internet links**: queue, progress, history, log, and downloader settings. A quick strip above the table covers short workflows.

## Terminal

Build a command for the bundled tools, browse hints, and keep a log. For a simple “download and lightly trim” workflow you often skip the terminal.

## Tooltips

Hover buttons and switches — many show a longer plain-language explanation.
