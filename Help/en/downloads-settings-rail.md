# Download settings (right rail)

Sections collapse individually. Main blocks:

- **Where to save** — folder on disk and filename pattern.
- **Format and quality** — curated combinations without a free-form “magic string”.
- **Subtitles and playlist** — pull subtitles or force a whole playlist.
- **Network** — speed cap and retry behaviour.
- **Cookies and sign-in** — Netscape cookie file or read from a browser profile when a site needs a session.
- **Extra arguments** — advanced users only; hints and a command preview are available.

## Cancel a download

The stop control sits **next to the URL field** for the active task. You do not need a duplicate stop button elsewhere on the rail.
