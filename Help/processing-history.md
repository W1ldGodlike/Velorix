# История обработки (правая панель FFmpeg)

Секция **«История обработки»** в правой панели редактора показывает завершённые экспорты ffmpeg, снимки кадров, авто-экспорт после загрузки и прогоны сценариев.

## Недельная сводка

Строка chips над списком — итоги за **7 суток**: всего записей, успехи, ошибки, отмены, суммарное время и число прогонов сценариев. Клик по chip с числом **фильтрует список** (кроме chip «Время»).

## Фильтры и поиск

- **Тип** — экспорт, кадры, авто-экспорт, пакет, сценарий.
- **Исход** — успех, ошибка, отмена.
- **Поиск** — путь файла, кодек, статус, id сценария, текст ошибки.

Список обновляется автоматически после новой записи (экспорт, сценарий и т.д.).

## Действия с записью

- **Повторить** — снова поставить тот же экспорт или снимок.
- **Повторить сценарий** — для записей типа «Сценарий»; если в сценарии задан URL, повтор идёт через загрузку yt-dlp.
- **Файл / Папка / Превью** — открыть выходной файл (если есть).
- **Исходник** — открыть входной файл в редакторе.

## Экспорт JSON

Кнопка **«Экспорт JSON»** сохраняет **видимые** записи (с учётом фильтров), недельную сводку и метаданные экспорта. Удобно для отчётов и поддержки.
