# Терминал, инспектор и планировщик

## Терминал (Инструменты)

Строка ввода с подсказками из `Data/ytdlp_commands.json` и `Data/ffmpeg_commands.json`. Разрешены только три префикса: **yt-dlp**, **ffmpeg**, **ffprobe** — приложение подставляет пути из настроек. При открытии подставляется превью команды с активной вкладки (скачивание или обработка).

Подробнее: [подсказки ffmpeg / yt-dlp](ffmpeg-terminal-hints.md) — JSON из `Data/*` и раздел «Встроенные сценарии» (`terminal-contract.ts`, локализация полей `summary`).

## Инспектор (Сервис через меню / ffprobe из очередей)

Отдельное окно для полного JSON **ffprobe** по выбранному файлу. Результат можно **скопировать в буфер** кнопкой «Копировать JSON».

## Планировщик watch folder (Сервис)

Указываются папка, файл сценария JSON и интервал проверки. Без папки и сценария старт недоступен — показывается подсказка.

## Проводник и нештатное закрытие

В **настройках** можно включить пункты контекстного меню Проводника (HKCU): **«Открыть в FluxAlloy»** (файл в очередь обработки как есть) и **«Быстро конвертировать в MP4»** — открывает FluxAlloy с пресетом H.264/AAC и автовыбором кодека.

Если прошлый сеанс завершился аварийно, при следующем запуске показывается напоминание: очереди уже подтянуты из `ui-session.json`.
