# Скачивание: буфер и перетаскивание

При открытии вкладки и при повторном фокусе приложение пробует добавить из буфера обмена любые строки вида `http(s)://`.

## Drag & Drop

На область прокрутки вкладки можно бросить:

- текст со ссылками (они будут найдены по пробелам/переводам строк);
- файл `.txt` или ярлык `.url` — будет прочитан текст и так же распарсен.

Превью команды считается для **первой** непустой строки в поле URL, как раньше.
