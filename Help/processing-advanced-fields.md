# Расширенная панель «Обработка»

Здесь отражены поля `FfmpegProcessingOptions` из ядра: обрезка `-ss`/`-to`, масштаб с `KeepAspect`, deinterlace (yadif/bwdif), EQ `brightness/contrast/saturation`, burn‑in субтитров, HDR→SDR через `HdrToSdr` (тонмаппер в цепочке zscale как в сборщике команд), LUT, шаблон выходного имени.

## Пресеты
Выпадающий список быстро подставляет кодеки и типичный CRF. После смены пресета при необходимости нажмите **Обновить превью команд**.

## Параллелизм
Поле задаёт число параллельных процессов ffmpeg. Для аппаратных кодировщиков NVENC ограничивайте сессию по возможностям драйвера.
