# Поле ссылки в редакторе (цепочка «скачать → обработать»)

Над таблицей очереди редактора есть блок для **одной или нескольких ссылок**. Он связывает загрузку и дальнейшие шаги без переключения на вкладку «Загрузки».

## Как пользоваться

1. Вставьте ссылки — берётся **первая непустая строка** как основная.
2. Включите **«Сначала скачать»**, если нужен обычный сценарий с сайта. Параметры совпадают с тем, что вы настроили на вкладке «Загрузки»: папка, формат, сеть, куки и дополнительные поля.
3. Если галочку снять, в очередь попадёт прямой адрес файла — это реже, но удобно для прямых ссылок на видео.
4. **«Затем запустить обработку»** ставит готовые файлы в очередь на перекодирование текущими настройками редактора.

## Повтор

Подходящие записи в журнале можно **повторить** — программа снова откроет этот блок с тем же адресом.
