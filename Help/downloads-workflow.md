# Вкладка «Загрузки»: очередь

![Схема очереди, журнала и панели настроек yt-dlp](assets/downloads-queue-overview.svg)

## Таблица

Каждая строка — один адрес. Колонки показывают ход загрузки: формат, размер, процент, скорость и **сколько осталось**. Справа от строки — значки: запуск, пауза (если система позволяет), повтор после ошибки, удалить, открыть файл или папку.

## Быстрая полоса

Над таблицей можно развернуть **короткую форму**: поле для ссылок и кнопка **«Скачать и добавить в редактор»** — программа сама поставит первую ожидающую строку в работу и по готовности откроет файл в превью редактора.

## Очередь целиком

Запуск по очереди идёт **сверху вниз**: пока одна строка активна, остальные ждут. Это снижает путаницу с несколькими загрузками сразу.

## История и журнал

Под таблицей — **история** (повторить ссылку одной кнопкой) и **живой журнал** текстом из программы-загрузчика. История умеет короткую сводку за последние дни, фильтр по исходу, **экспорт видимых записей в JSON** и обновление списка после завершения загрузки без ручного «Обновить».
