# «О программе», журналы и поддержка

## Версия

В диалоге «О программе» видно номер сборки приложения и версии встроенного движка страницы — это помогает при обращении в поддержку.

## Папка журналов

Там же есть переход к **папке журналов**. Файлы небольшие: при необходимости их можно приложить к письму.

## Архив для поддержки

Кнопка **архива** собирает несколько служебных файлов в один zip — удобно отправить целиком, не выбирая вручную. После ручного packaged smoke в архиве есть блоки `winPackagedSmoke:`, `linuxPackagedSmoke:`, `macosPackagedSmoke:` (чеклисты из настроек) — см. [packaged-windows-smoke.md](packaged-windows-smoke.md), [packaged-linux-smoke.md](packaged-linux-smoke.md), [packaged-macos-smoke.md](packaged-macos-smoke.md).

## Временные файлы

Отдельно показывается оценка **временных** данных (превью, незаконченные загрузки, старые служебные папки). Очистка идёт в два шага, чтобы случайно не удалить нужное.
