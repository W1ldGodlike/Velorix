# FluxAlloy — быстрый старт

1. Откройте вкладку «Скачивание».
2. Вставьте URL (по одному на строку).
3. Нажмите «Добавить в очередь», затем «Старт очереди».

Для обработки локальных файлов откройте вкладку «Обработка», добавьте файлы и нажмите «Запустить пакет». Шаблоны под соцсети описаны в [processing-social-presets.md](processing-social-presets.md). Терминал, инспектор и watch folder — [tools-terminal-inspector.md](tools-terminal-inspector.md). Про кодеры GPU/CPU см. [hardware-encoding.md](hardware-encoding.md).
