Velorix — иллюстрации для базы знаний
=====================================

Сюда кладите картинки (PNG, JPG, WebP, GIF, SVG), на которые ссылаются статьи в Markdown так:

  ![Подпись](assets/имя-файла.png)

Путь всегда начинается с "assets/". Файлы попадают в установку вместе с папкой Help.

Реальные скриншоты окна лучше делать вручную при стабильной вёрстке: подставьте файл сюда и обновите ссылку в статье.

Стабильные SVG-схемы (не фотоскриншоты, не устаревают от пикселей темы):
  workspace-tabs-diagram.svg      — вкладки Редактор / Загрузки / Терминал
  downloads-queue-overview.svg    — очередь загрузок и rail настроек
  editor-preview-timeline.svg     — превью, транспорт, таймлайн, FFmpeg
  knowledge-dialog-toc.svg        — окно справки и оглавление FAQ

Фотоскриншоты PNG/WebP при необходимости добавляйте вручную и ссылайтесь так же:
  ![Подпись](assets/имя.png)
