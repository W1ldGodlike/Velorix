# Логи и диагностика

## Где лежат логи

Файлы Structured JSON (Serilog Compact): папка `%LocalAppData%\FluxAlloy\logs\`.

Имя файла за запуск: `FluxAlloy-ГГГГММДД-ВРЕМЯ-PID.json`. При превышении размера включается ротация средствами Serilog.

## В настройках

- **Подробные логи (Verbose)** — пишет избыточность для отладки (включите при обращении в поддержку или по инструкции).
- **Открыть папку с логами** — переход в каталог в проводнике Windows.

## Экспорт для поддержки

В меню «Справка» → «О программе» используйте экспорт лога (упаковка последних сессий в ZIP рядом с логами).
