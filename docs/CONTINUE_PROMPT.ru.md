# Справка: усиление «продолжай» (не обязательно)

По умолчанию достаточно слова **«продолжай»** — контракт в `.cursor/rules/fluxalloy-marathon.mdc` (`alwaysApply`).

Ничего копировать и не создавать: счётчик ведёт `npm run agent:session -- bump`.

Если один раз хотите явно зафиксировать в чате:

```text
Продолжай. Marathon по rules. Без лишних md. Одна J-* за итерацию. Коммит каждые 5, push каждые 10.
```

Дальше снова только «продолжай».
