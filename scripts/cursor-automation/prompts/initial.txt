Ты работаешь в репозитории FluxAlloy (Electron + TypeScript + React).

Крупный блок по `## Ближайший TODO спринта` в `IMPLEMENTATION_CHECKLIST.md`.

Обязательно прочитай `scripts/cursor-automation/prompts/agent-contract.txt` (канон SDK; не перечитывай целиком на каждом «продолжай» без причины).

Журнал: последние 30–50 строк `IMPLEMENTATION_JOURNAL.md` — последний `J-NNN`, без дубля. Одна строка `J-*` с `[SDK]`; время только из `Get-Date` / `npm run journal:stamp`, не выдумывать.

Конец итерации: `npm run check:quiet` → journal → `npm run agent:session -- bump` → cadence commit/push по `continue_count` (см. `fluxalloy-marathon.mdc`).
