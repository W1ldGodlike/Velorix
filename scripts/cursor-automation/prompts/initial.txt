Ты работаешь в репозитории FluxAlloy (Electron + TypeScript + React).

Задача этой итерации:
1. Сначала прочитай `scripts/cursor-automation/prompts/agent-contract.txt` и следуй ему как обязательной инструкции для этой и следующих итераций.
2. Открой IMPLEMENTATION_CHECKLIST.md и IMPLEMENTATION_JOURNAL.md — выбери следующий логичный блок незакрытых пунктов со статусами [ ] или [~].
3. Реализуй крупным осмысленным куском. Не трогай FLUXALLOY_TZ.md без явной просьбы пользователя в прошлой переписке.
4. В конце выполни `npm run check` из корня репозитория; при ошибках исправь.
5. При необходимости обнови только эти два файла: IMPLEMENTATION_CHECKLIST.md (статусы) и IMPLEMENTATION_JOURNAL.md (одна строка с датой/временем, источником `[SDK]` и сутью — без простыней).
   ВАЖНО: время для IMPLEMENTATION_JOURNAL.md нельзя придумывать. Перед записью обязательно выполни команду получения текущего локального времени (`Get-Date -Format "yyyy-MM-dd HH:mm:ss"` в PowerShell или эквивалент) и используй ровно её результат.
   ВАЖНО: ты SDK-агент, поэтому в журнале пиши только `[SDK]`, никогда `[Assistant]`. `[Assistant]` — только для обычного чата с пользователем.
6. Если всё ок — один коммит с понятным сообщением на русском.

Отчитайся кратко: что сделано, что блокирует следующий шаг (если есть).
